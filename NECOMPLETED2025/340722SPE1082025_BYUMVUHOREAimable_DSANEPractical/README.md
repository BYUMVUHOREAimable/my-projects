# Road Management System

## Project Overview
This is a C++ implementation of a Road Management System that helps manage and analyze road networks between cities. The system provides functionality to handle road connections, city management, and network analysis.

## Features
- City management (add, remove, list cities)
- Road network management (add, remove, list roads)
- Network analysis capabilities
- File-based data persistence
- Interactive command-line interface

## Project Structure
- `road_management_system.cpp` - Main source code file
- `cities.txt` - Data file storing city information
- `roads.txt` - Data file storing road network information
- `FlowChart.png` - Visual representation of the system's workflow

## Prerequisites
- C++ compiler (supporting C++11 or later)
- Windows operating system (for the executable)
- Visual Studio Code (recommended IDE)

## Building the Project
1. Clone this repository
2. Open the project in Visual Studio Code
3. Compile the source code:
   ```bash
   g++ road_management_system.cpp -o road_management_system
   ```
4. Run the executable:
   ```bash
   ./road_management_system
   ```

## Usage
1. Launch the program using the executable
2. Follow the interactive menu to:
   - Manage cities
   - Manage roads
   - Analyze the network
   - View system information

## Data Files
- `cities.txt`: Stores the list of cities in the system
- `roads.txt`: Stores the road connections between cities
